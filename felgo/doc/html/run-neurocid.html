Chapter 2: Running the Neurocid-Felgo frontend

The whole idea behind Neurocid is to train a swarm of "intelligent" space ships to fight another space ship team. There are two things required to be able to train a swarm:

  • A curriculum (a list of scenarios to train with one after the other) which can be defined by creating scenarios (either in C++ or as json file) and providing placers (Neurocid/lua/placer/)
  • A fitness function (the default fitness function resides in Neurocid/lua/fitness/amir.lua)

Going into detail on how to do that exactly is out of scope for this tutorial but there is a shell script which provides everything required to run a curriculim. To run the training do the following:

cd Neurocid/
felgo/training.sh
